# Calculator

Un piccolo progetto di esempio per imparare il flusso di lavoro con Git e GitHub.

Contiene una semplice calcolatrice in Python con qualche operazione di base.

## Uso

```python
from calculator import add, subtract

add(2, 3)       # 5
subtract(5, 2)  # 3
```

## Test

```bash
python -m pytest
```
